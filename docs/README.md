# Website (GitHub Pages)

This folder contains the static site for FMoIP.

**To enable the website:** In the repository go to **Settings → Pages**. Under "Build and deployment", set **Source** to **Deploy from a branch**. Choose your default branch (e.g. `main`) and set the folder to **/docs**. Save. The site will be available at `https://<username>.github.io/<repo-name>/`.

- **Main page:** `index.html`
- **Privacy policy:** `privacy-policy.html` (linked from the main page)
